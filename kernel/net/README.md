# T.K. Kernel Network Stack Module

## Overview

Implements the network protocol stack.

## Key Components

- `tuesk_socket.c` - Socket implementation
- `tuesk_tcp.c` - TCP protocol
- `tuesk_udp.c` - UDP protocol
- `tuesk_ip.c` - IP layer
- `tuesk_eth.c` - Ethernet driver interface

## Features

- BSD socket interface
- TCP/IP protocol stack
- Network driver integration
- Packet routing
- Network statistics
