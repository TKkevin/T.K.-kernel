# T.K. Kernel Process Management Module

## Overview

Manages process and thread creation, scheduling, and lifecycle.

## Key Components

- `tuesk_process.c` - Process management
- `tuesk_thread.c` - Thread management
- `tuesk_scheduler.c` - Task scheduler
- `tuesk_context.asm` - Context switching assembly

## Features

- Process creation and termination
- Thread support
- Priority-based scheduling
- Context switching
- Process synchronization
