# T.K. Kernel IPC Module

## Overview

Provides inter-process communication mechanisms.

## Key Components

- `tuesk_ipc.c` - IPC core
- `tuesk_pipe.c` - Pipe implementation
- `tuesk_msgqueue.c` - Message queue
- `tuesk_shm.c` - Shared memory

## Features

- Message passing
- Pipes and named pipes
- Message queues
- Shared memory regions
- Synchronization primitives
