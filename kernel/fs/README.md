# T.K. Kernel File System Module

## Overview

Implements the virtual file system and disk I/O abstraction.

## Key Components

- `tuesk_fs.c` - File system core
- `tuesk_vfs.c` - Virtual file system layer
- `tuesk_inode.c` - Inode management
- `tuesk_bio.c` - Block I/O

## Features

- Virtual file system
- Inode-based file operations
- Block device abstraction
- File permissions and attributes
- Directory operations
