## Design Guidelines

1. Use TuesdayOS brand colors listed above
2. Style: Modern, clean, professional with gradient backgrounds and subtle geometric patterns
3. TuesdayOS logo should be prominent but not overwhelming
4. Must provide variants for 4-bit, 8-bit, 16-bit, and 32-bit color modes
5. BMP format for system use, JPG format for user choice
