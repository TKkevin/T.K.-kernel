# TuesdayOS System Icons

## Required Icon Sets

All icons should be provided in ICO format with multiple sizes:
16x16, 32x32, 48x48, and 256x256 (where applicable).

### OS Branding Icons
- tuesdayos.ico — TuesdayOS logo
- tuesk.ico — T.K. Kernel icon

### Desktop Icons
- my_tuesday.ico — My Tuesday (computer icon)
- tuesplorer.ico — Tuesplorer file manager

### Drive Icons
- drive_hd.ico — Hard disk drive
- drive_cd.ico — CD/DVD drive
- drive_net.ico — Network drive

### Folder Icons
- folder.ico — Standard folder
- folder_open.ico — Open folder

### File Type Icons
- file_txt.ico — Text file
- file_doc.ico — Document file
- file_exe.ico — Executable file

### System Icons
- settings_panel.ico — Settings Panel (Control Panel)
- network.ico — Network
- internet.ico — Internet
- hardware_mgr.ico — Hardware Manager
- recycle_full.ico — Recovery Bin (full)
- recycle_empty.ico — Recovery Bin (empty)

### Security Icons
- guard.ico — TuesdayOS Guard (security center)
- firewall.ico — Firewall
- lock.ico — Lock/security

### Media Icons
- media_player.ico — TuesMedia Player
- audio.ico — Audio file
- video.ico — Video file

### Application Icons (replacing Windows apps)
- tuesdaypad.ico — TuesdayPad (notepad)
- tuesdaycalc.ico — TuesdayCalc (calculator)
- tuesdaydraw.ico — TuesdayDraw (paint)
- tuesdaywrite.ico — TuesdayWrite (wordpad)
- tuesbrowser.ico — TuesBrowser (IE)
- tuesmedia.ico — TuesMedia (WMP)
- glyphmap.ico — Glyph Map (charmap)
- audiorec.ico — Audio Recorder
- tueshearts.ico — Hearts game
- tuessol.ico — Solitaire
- tuesfreecell.ico — FreeCell
- tuesmine.ico — Minesweeper

## Design Guidelines

1. Use TuesdayOS brand colors: Tuesday Blue (#4A90D9), Tuesday Green (#2ECC71), Tuesday Gold (#F39C12)
2. Style should be consistent across all icons
3. Provide all sizes in single ICO file
4. Maintain visual clarity at 16x16 size
