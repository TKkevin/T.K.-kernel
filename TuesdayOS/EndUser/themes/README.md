# TuesdayOS Themes

## Default Theme: Tuesday Blue

The default TuesdayOS theme uses the following design tokens:

| Element | Color | Hex |
|---|---|---|
| Window Title Bar | Tuesday Blue | #4A90D9 |
| Window Title Text | White | #FFFFFF |
| Window Background | Tuesday Light | #ECF0F1 |
| Button Face | Tuesday Light | #ECF0F1 |
| Button Text | Tuesday Dark | #2C3E50 |
| Selection | Tuesday Blue | #4A90D9 |
| Selection Text | White | #FFFFFF |
| Desktop | Tuesday Dark | #2C3E50 |
| Taskbar | Tuesday Dark | #2C3E50 |
| Start Button | Tuesday Gold | #F39C12 |

## Theme Variants

1. **Tuesday Blue** (default) — Blue gradient, professional look
2. **Tuesday Dark** — Dark theme for low-light environments
3. **Tuesday Light** — Light, airy theme
4. **Tuesday Classic** — Retro-inspired, minimal theme

## Theme File Structure

Each theme should include:
- .theme file (INI-format theme descriptor)
- .msstyles equivalent (visual style definition)
- Color scheme definitions
- Font specifications
- Sound scheme references
- Screensaver references
