# TuesdayOS Source Attribution

## Origin

TuesdayOS is derived from the Windows XP Service Pack 1 (NT 5.1) source
code originally published in the Source/XPSP1/NT directory of this
repository. The original source code was created by Microsoft Corporation
and made available under separate license terms.

The original Windows XP SP1 (NT 5.1) source code upon which TuesdayOS
