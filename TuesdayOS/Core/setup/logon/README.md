# TuesdayOS Logon Screen Resources

This directory contains graphics and resources for the TuesdayOS
logon screen, replacing the original Windows XP logon screen resources.

## Required Files

| File | Description | Format | Size |
|---|---|---|---|
| logon_default.bmp | Default logon screen background | BMP | 1024x768 |
| logon_server.bmp | Server edition logon background | BMP | 1024x768 |
| user_icon_default.ico | Default user account icon | ICO | multi-size |
| user_icon_admin.ico | Administrator user icon | ICO | multi-size |

## Design Guidelines

1. Logon background should use TuesdayOS brand colors with a clean, modern look
2. Must work at multiple resolutions (800x600, 1024x768, 1280x1024, 1920x1080)
3. User icons should be provided in 48x48 and 256x256 sizes
4. Background must be compatible with the logon UI overlay elements
