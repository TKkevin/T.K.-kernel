# TuesdayOS Shell Graphics (TuesShell32)

This directory contains graphics used by the TuesShell32 component,
including the About TuesdayOS dialog graphics.

## Required Files

| File | Description | Format | Size |
|---|---|---|---|
| about_tuesdayos.bmp | About TuesdayOS dialog graphic | BMP | variable |
| about_tuesdayos_16.bmp | 16-color About box version | BMP | variable |
| about_tuesdayos_256.bmp | 256-color About box version | BMP | variable |
| eula_graphic.bmp | EULA dialog graphic | BMP | variable |

## Design Guidelines

1. About box graphic should prominently feature the TuesdayOS logo
2. Provide color-depth variants for compatibility
3. Use TuesdayOS brand colors
4. Keep file sizes reasonable for dialog display
