# TuesdayOS Utilities

Command-line tools and system utilities for TuesdayOS.

## Utilities

- **tkshell** - Command shell
- **tkls** - List files
- **tkcat** - Display files
- **tkmkdir** - Create directories
- **tkps** - Process listing
- **tkkill** - Terminate processes

## Utility Naming Convention

- Prefix: `tk`
- Example: `tklist` for listing utility

## Building Utilities

```bash
cd utilities
make all
```
